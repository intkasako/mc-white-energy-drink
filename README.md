## White Energy Drink 
> A small mod experiment exploring custom items and gameplay tweaks.

A simple Minecraft mod built with Fabric that adds a custom energy drink item to the game.

## Features

- Adds a new craftable item
- Custom item effects
- Custom crafting recipe
- Built for Minecraft 1.21.1 (Fabric)

## Development

- Minecraft: 1.21.1
- Loader: Fabric
- Java: 21
